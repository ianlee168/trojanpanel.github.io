# Cloudflare

Cloudflare: https://www.cloudflare.com/zh-cn/

注意:

  1. 节点端口必须使用CF指定的端口 (opens new window)

  2. Cloudflare 代理状态必须设置为已代理，SSL/TLS加密模式必须设置为完全或者完全（严格）

  1. 如果前端开启了https，则必须使用CF指定的https端口 (opens new window) ，http同理，否则无法通过域名访问面板
