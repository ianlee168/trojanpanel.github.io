# v2rayN配置Hysteria（Windows）

  1. 下载v2rayN (opens new window)

  2. 下载Hysteria (opens new window)

  3. 将hysteria执行文件复制到v2rayN-Core目录下

  1. 编辑hysteria配置文件
{
  "server": "域名:端口",
  "protocol": "udp",
  "up_mbps": 100,
  "down_mbps": 100,
  "auth_str": "密码",
  "socks5": {
    "listen": "127.0.0.1:1080"
  }
}

备注：auth_str不需要base64编码

  1. 开启对应的udp端口或者关闭防火墙

  2. 打开v2rayN，添加自定义服务器

  3. 将v2rayN的系统代理设置为自动配置系统代理

  4. 选中节点，点击Enter即可连接节点
